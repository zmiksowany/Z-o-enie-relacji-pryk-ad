Przykład złożenia relacji

1. Definicja złożenia relacji:

Złożeniem (superpozycją) relacji R ⊂ A × B i S ⊂ C × D nazywamy relację S ∘ R ⊂ A × D spełniającą warunek:

(x, y) ∈ S ∘ R ⊂ A × D ⇐⇒ ∃t ∈ B ∩ C : ( (x, t) ∈ R ∧ (t, y) ∈ S ).

2. Zbiory:

  • A = {a,b}
  • B = {c,d}
  • C = {d,e}
  • D = {f,g}

3. Relacje:

    Dla uproszczenia załóżmy: R=A × B, S=C × D, wtedy:

  • R = {(a,c), (a,d),(b,c), (b,d)}
  • S = {(d,f), (d, g), (e,f), (e,g)}

4. Złożenie relacji (S ∘ R):

  • B ∩ C={d}. Czyli szukamy w R par (x,d) oraz w S par (d,y).

5. Ostateczne złożenie:

S ∘ R = {(a,f), (a,g), (b,f), (b,g)}
